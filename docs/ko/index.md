# RAG 앱 체크리스트

RAG 앱을 로컬 프로토타입에서 프로덕션으로 배포하기 위한 두 가지 가이드입니다.

---

## [RAG 앱 프로덕션 가이드](ship.md)

프로덕션 배포 시 자주 발생하는 문제들에 대한 복붙 가능한 해결 코드입니다:

- **환각 감지** — RAGAS를 활용한 10줄 자동 검증
- **입력 안전** — 프롬프트 인젝션과 개인정보 보호를 하나의 함수로
- **비용 모니터링** — 요청당 비용 로깅과 모델 라우팅
- **검색 디버깅** — 로깅, 유사도 임계값, 진단 방법
- **청킹** — 권장 설정과 품질 확인 방법
- **스트리밍** — FastAPI와 바닐라 JS로 SSE 구현
- **에러 처리** — rate limit 및 컨텍스트 초과 대응 재시도 로직

## [RAG 앱 테스트 가이드](test.md)

LLM 출력은 비결정적이므로 일반적인 assert를 적용하기 어렵습니다. 대신 사용할 수 있는 방법들입니다:

- **검색 테스트** — 올바른 컨텍스트가 검색되는지 확인 (결정적, 무료)
- **생성 테스트** — 답변의 근거 확인 및 환각 감지 (RAGAS)
- **안전 테스트** — 프롬프트 인젝션과 개인정보 처리를 parametrize로 검증
- **에이전트 테스트** — 도구 선택 정확도, 스텝 효율성, 환각률 측정
- **LLM 목킹** — 빠르고 비용 없는 테스트 실행
